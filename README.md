## Notes App 📝

A full-stack Notes Application where users can create, update, and delete notes.
This project demonstrates integration between frontend and backend using REST APIs.

## Features

* Add new notes
* Edit existing notes
* Delete notes
* Frontend and backend integration

## Tech Stack

**Frontend**

* HTML
* CSS
* JavaScript

**Backend**

* Node.js
* Express.js

## How It Works

1. User interacts with the frontend interface.
2. Frontend sends HTTP requests to backend APIs.
3. Backend processes the request and manages the notes data.
4. The response is sent back to the frontend and displayed in the UI.
